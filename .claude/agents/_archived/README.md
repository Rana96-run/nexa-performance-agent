# _archived/ — retired agent definitions

When a role is retired or replaced, move its `.md` here instead of deleting it —
so the history of who-existed stays readable and the agent can be revived. Files
here are **not** active agents (Claude Code ignores the `_archived/` subfolder).

## Convention
- `git mv .claude/agents/<role>.md .claude/agents/_archived/<role>.md`
- Add a one-line note below: date · why retired · what replaced it.
- Update `docs/_shared/org-chart.md` and `.claude/agents/README.md`.
