# Playbook — CRO Paid Specialist

**Seat:** Performance Marketing. **Agent:** `cro-paid-specialist`.

## Purpose
Decide whether a CPQL problem is the ad or the page, and spec the page fix.

## Procedure
1. Confirm traffic is healthy (impressions/clicks fine) but conversion is weak —
   else it's a media-buyer / keyword problem, hand back.
2. Diagnose the page: bounce, scroll depth, form-field drop-off, LP↔offer↔keyword match,
   load speed, mobile/RTL rendering.
3. Write the **LP spec**: headline/copy, section order, form fields, trust microcopy,
   CTA treatment — Arabic in MSA, RTL layout.
4. Open an Asana `[Creative Brief]` for the Landing Page Agent (`D:\Landing Page Agent`)
   with: current CPQL, target, page URL, exact change, window (explicit dates).
5. After the build ships, ask `paid-media-analyst` to re-measure CPQL at 7d/14d.

## Rules
You spec and brief; you never build pages or change ad budgets. Reference
`.claude/skills/cro-paid-specialist.md` for the LP↔CPQL loop detail.

## Write to memory
LP patterns that moved CPQL → `memory/agents/performance-marketing/cro-paid-specialist/`.

## Done means
A spec + an Asana brief with the full context, handed off. Re-measure scheduled.
