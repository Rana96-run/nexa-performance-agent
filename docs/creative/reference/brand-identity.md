# Qoyod Brand Identity — Complete Visual Specs

Extracted from Qoyod Brand Guidelines V3 (July 2024) + Design System for Claude (April 2026).
This is the single source of truth for all visual decisions.

---

## 1. Logo System

### Main Logo: "QOYOD" + Q Brandmark
- Typography-based wordmark with Q circle + small square
- Minimum size: 15mm
- Clear space: 4x square unit around logo
- Always use both elements together — never separate

### Sub-Brand Logos
Each sub-brand logo = Q Brandmark + sub-brand name (Arabic + English) + QOYOD logo
They appear together separated by a vertical line: `[Sub-brand logo] | QOYOD`

| Sub-brand | Arabic Name | English Name |
|-----------|------------|-------------|
| QBookkeeping | مسك الدفاتر | BOOKKEEPING |
| QFlavours | فليفرز | Flavours |
| QTahseel | تحصيل | COLLECTION |
| QLend | ليند | LEND |
| QAcademy | أكاديمي | ACADEMY |

### Logo Placement Rule
- **Logo/sub-logo:** Bottom-right corner
- **qoyod.com:** Bottom-left corner
- Both sit on the bottom row of the layout grid

### Logo Don'ts
- No distortion, rotation, shadows, filters, outlines, boxes, or non-brand colors

### Gradient Rules
- **Direction:** Always 45° from top-right to bottom-left
- **Dark gradient:** Navy `#021544` → Turquoise/Cyan
- **Light gradient:** Turquoise/Cyan → White `#FFFFFF`
- **Header gradient:** Used on dark backgrounds for headlines

### Color Balance (for social media designs)
| Element | Dark BG Layout | Light BG Layout |
|---------|---------------|-----------------|
| Background | 40% Navy | 40% White/Cyan light |
| Content | 30% | 35% |
| Header/Headline | 20% | 20% |
| Accent/Other | 10% | 5% |

**QBookkeeping specific:** Navy background dominant + Orange for headlines, CTAs, checkmarks, highlighted words

## 3. Typography

### Font: Lama Sans (ONLY font — Arabic AND English)

| Weight | Usage in Designs | Arabic | English |
|--------|-----------------|--------|---------|
| **Black** | Display headlines (social media) | ✓ | ✓ |
| **Bold** | Headlines, CTA buttons, stat values | ✓ | ✓ |
| **SemiBold** | Card titles, eyebrow pills, buttons | ✓ | ✓ |
| **Medium** | Sub-headlines, body emphasis | ✓ | ✓ |
| **Regular** | Body copy, captions, bullet points | ✓ | ✓ |

### Social Media Typography Sizes (approximate for 1080px width)
| Element | Weight | Relative Size |
|---------|--------|--------------|
| Main headline | Black | Largest — fills 30-40% of width |
| Sub-headline | Medium | 40-50% of headline size |
| Body/bullets | Regular | Standard reading size |
| CTA button | Bold | Medium — readable in button |
| Logo text | Per brand | Small — bottom corner |

### Typography Rules
- Arabic text: **always right-aligned** (محاذاة يمين)
- English text: left-aligned
- Extended/stretched letters for dramatic effect: "تعطّـــل" "نمـــوك" "كثيـــــرة؟"
- Headlines: max 2 lines, 6-8 words per line
- One keyword per headline gets the accent color treatment

## 4. Graphic Elements

### Q Key Element (Decorative)
The Q brandmark shape (circle + small square) used as background decoration:
- **Soft Light version:** 50% opacity, in brand color
- **Outline version:** Thin stroke, no fill
- Use sparingly — watermark or background texture
- Never competes with main content

### Concentric Circles
- Appear as subtle background element (seen in QBookkeeping and QFlavours campaigns)
- Opacity: ~15%
- Centered behind the main visual element or offset
- Creates depth without distraction

### Checkmarks (✓)
- Used for feature lists and benefits
- Color matches sub-brand accent (Orange for QBookkeeping)
- Circular background optional (seen in some campaign variants)

## 5. Layout Grid (Social Media — Story 1080×1920)

### Element Positioning
| Zone | Content | Position |
|------|---------|----------|
| **Top third** (0-640px) | Headline + sub-headline | Right-aligned |
| **Middle third** (640-1280px) | Main visual / feature list / stats | Center or right |
| **Bottom third** (1280-1920px) | CTA + visual element + logos | CTA center, logos bottom |
| **Bottom-right** | Sub-brand logo ∣ QOYOD logo | Fixed position |
| **Bottom-left** | qoyod.com | Fixed position |

### Approved Layout Types
1. **Text + Device Mockup** — Headline top, device (laptop/tablet/phone) showing UI in middle-bottom
2. **Text + Person** — Headline top, Saudi person in bottom half
3. **Text + Feature List** — Headline top, checkmark list in middle, device small
4. **Split Screen** — Left/right comparison (before/after)
5. **Text Only** — Bold headline taking most of the space, minimal supporting elements
6. **Text + Stats** — Headline top, large numbers/statistics in middle

## 7. Tone of Voice

| Trait | How it shows in content |
|-------|------------------------|
| **Engaging** | Questions in headlines, relatable scenarios |
| **Confident** | Bold claims backed by credentials (SOCPA, ZATCA) |
| **Professional & Clear** | Clean language, no jargon overload |
| **Fazeea (فزيع)** | "نشتغل عشان ما تشتغل" — going beyond expectations |

### Language register by platform
| Platform | Register |
|----------|---------|
| LinkedIn | فصحى بسيطة — professional |
| Instagram | Mix فصحى + لهجة سعودية |
| TikTok/Snapchat | لهجة سعودية dominant |
| X (Twitter) | Short, punchy, can be dialectal |

## 8. Pre-Flight Checklist

Before delivering any design prompt, verify:
- [ ] Correct brand identified and accent color used
- [ ] Hex codes specified (not just color names)
- [ ] Font: Lama Sans with weight specified for every text element
- [ ] Arabic text right-aligned
- [ ] Layout follows grid: headline top-right, visual bottom-left, logo bottom-right
- [ ] Q graphic element described (50% opacity soft light or outline)
- [ ] Concentric circles if applicable (15% opacity)
- [ ] Gradient direction: 45° top-right to bottom-left
- [ ] CTA button with accent color + white text
- [ ] Sub-brand logo + QOYOD logo — bottom-right
- [ ] qoyod.com — bottom-left
- [ ] Tone matches brand voice
