## 1. Brand Foundation

3. **Sustainability & Growth** — Long-term impact

5. **Innovation** — Forward-thinking solutions

### Brand Essence
- Born 2016, Saudi cloud accounting
- ZATCA accredited
- No downloads needed — any device, any browser
- Quick, easy, fully secure

## 2. Logo & Brandmark

### Primary Logo
- Typography-based logo focusing on "Qoyod" name
- Colors combine technology + company mission
- Balanced, organized, flexible

### Q Brandmark
- The "Q" letter used as a shortened mark
- Contains a square shape element
- Used separately across applications

### Logo Rules
- **Minimum size:** 15mm
- **Clear space:** 4x square shape width around logo
- **Construction:** Balanced between constant x spaces

### Logo Misuse (NEVER DO)
- Do NOT distort relationship between word and square
- Do NOT distort the logo or rotate it
- Do NOT change proportions
- Do NOT use one element alone (word or square must be together)
- Do NOT use different font
- Do NOT add shadow or filters
- Do NOT put logo inside a box
- Do NOT use colors outside the guideline
- Do NOT use outline on brandmark

### Background Variations
- Logo has black and white versions
- Must appear in correct color on different backgrounds

### Sub-Logos (5 Products)
Each product has its own square shape color:
1. **Qoyod** (main) — Primary Navy/Turquoise
2. **QTahseel** — Green (collection service + QTahseel Plus)
3. **QLend** — Blue (lending)
4. **QAcademy** — Orange (education)
5. **QBookkeeping** — Purple (bookkeeping service)

## 4. Typography

### Font Family
**Lama Sans** — Used for BOTH Arabic and English

### Font Weights
| Weight | Usage |
|--------|-------|
| **Bold** | Headlines |
| **Medium** | Subheadline 1 & 2 |
| **Regular** | Body copy, Caption, Folios |
| **Black** | Special emphasis (sparingly) |

### Font Styling Rules
- All Arabic copy must be **right-aligned**
- Headlines: **Bold**
- Subheadline 1: **Medium**
- Subheadline 2: **Medium**
- Body copy: **Regular**
- Caption: **Regular**
- Folios: **Regular**
- Keep type sizes and leading consistent within similar communications

## 5. Graphic Elements

### Icons
- Flat style icons
- One or two primary/secondary colors only
- Uniform line width
- Used at small to medium sizes

### Graphics Key Element (Square Shape)
- Can be used as watermark or page divider
- Two styles:
  - **Soft Light:** 50% opacity colors
  - **Outline:** Shape outline only
- Use sparingly

### Layout Grid System (Key Element Positioning)
| Element | Position |
|---------|----------|
| **Text** | Upper-right corner |
| **Photos/Vector/3D** | Lower-left corner |
| **Logo/Sub-logos** | Lower-right corner |
| **Website URL** | Lower-left corner |

### Key Element with Text
- Maximum 3 short lines (slogan, naming, services)
- Text color = same as square color in key element
- Both Arabic and English text on the left
- Positioned between the underneath of the O letter lines

### Design Composition Types
1. Only Text
2. Text with Element
3. Text with Real Photo
4. Text with Vector
5. Text with 3D
6. Text with Real Photo Inside the Box

## 8. Applications Checklist

### For ANY Design Output, Verify:
- [ ] Logo used correctly (no misuse violations)
- [ ] Clear space around logo maintained (4x square)
- [ ] Colors from approved palette only
- [ ] Gradient at 45 degrees (top-right to bottom-left)
- [ ] Font is Lama Sans (correct weight for element type)
- [ ] Arabic text right-aligned
- [ ] Color balance follows the percentage rules
- [ ] Layout follows grid: text upper-right, visuals lower-left, logo lower-right
- [ ] Key element (square) used sparingly, correct opacity
- [ ] Sub-product uses its designated color
- [ ] Tone matches: engaging, confident, professional, fazeea
- [ ] No unauthorized filters, shadows, outlines, or effects
